# bento_post

## 概要
弁当注文用のスクリプトです．
GASによって書かれています．

基本的なコマンドを以下に示します．

## 使い方

`order [option] [value]`

### options

 `-d MMDD [ifn]`
 MM月DD日の注文を設定する

 `-w [ifn]{6}`
 次週(Mon~Sat)の注文を設定する

 `-a`
 自動注文設定(カレンダーの昼食会に合わせて自動設定)をON/OFFする
